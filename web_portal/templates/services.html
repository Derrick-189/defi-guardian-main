Frequently Asked Questions

Can I self-host DeFi Guardian?

Yes. DeFi Guardian is open-source. Clone the repository and follow the setup guide to run it on your own infrastructure.

Do I need to install the verification tools separately?

Tools not installed locally will run in simulated mode, producing realistic deterministic output. Install the binaries to use real verification.

Is my contract source code stored?

Contracts are stored locally in your instance's database. When using the hosted service, data is encrypted at rest and never shared with third parties.
