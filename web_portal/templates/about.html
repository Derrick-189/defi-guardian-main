⛨ About DeFi Guardian

A unified formal verification suite for smart contracts and DeFi protocols.

What is DeFi Guardian?

DeFi Guardian is an open-source formal verification platform that integrates eight industry-leading verification engines into a single, developer-friendly interface. It enables smart contract developers and security researchers to apply mathematical proof techniques to their code — catching bugs that testing alone cannot find.

Unlike fuzzing or unit testing, formal verification exhaustively explores all possible execution paths and proves (or disproves) correctness properties. DeFi Guardian makes this power accessible through a web portal, REST API, and desktop application.

The 8 Verification Tools

Tool Type Language Technique Best For
SPIN Model Checker Promela LTL model checking Concurrency, protocol verification
Coq Proof Assistant Gallina / Coq Dependent type theory Mathematical invariants, full correctness
Lean 4 Proof Assistant Lean 4 Dependent type theory Readable proofs, Mathlib integration
Certora Formal Verifier Solidity / CVL SMT-based verification EVM smart contracts, DeFi protocols
Kani Model Checker Rust Bounded model checking (CBMC) Rust smart contracts, off-chain code
Prusti Deductive Verifier Rust Viper-based verification Pre/post conditions, loop invariants
Creusot Deductive Verifier Rust Why3 / SMT Functional correctness of Rust code
Verus Verification System Rust SMT-based (Z3) Systems code, low-level Rust

How It Works

  1. 1
    Upload Contract

    Upload your Solidity (.sol), Rust (.rs), or Promela (.pml) source file via the web portal or API.

  2. 2
    Select Tools

    Choose one or more verification engines. Each tool targets different properties and languages.

  3. 3
    Write Specifications

    Define LTL properties or CVL rules. Use the LLM generator to translate plain-English descriptions.

  4. 4
    Run Verification

    DeFi Guardian invokes the selected tools and streams live output to your dashboard via WebSocket.

  5. 5
    Analyse Results

    Explore counterexamples, replay execution traces, and visualise the state space interactively.
